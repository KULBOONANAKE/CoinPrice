### Features
* Display live price for each cryptocurrency
* Chart for analytics

### Tech!

* MVVM Architecture with SwiftUI
* Build a complete iOS mobile application with SwiftUI with networking layer
* Fetching live cryptocurrency data from an API
* Dark mode support
